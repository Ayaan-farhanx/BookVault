### BookVault
BookVault is a responsive and interactive library management application built with HTML, CSS and JavaScript. It allows users to organize, track and manage their personal book collection with features such as adding, removing, searching, filtering, and sorting books.

## Features
- Add and remove books
- Toggle reading status
- Search books
- Filter by read/unread status
- Sort by title/author/pages
- Save library data with localStorage

## Technologies
- HTML5
- CSS
- JavaScript
